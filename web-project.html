Website

1. Preparation

To create our website, we began by working on individual parts to refresh and familiarize ourselves with HTML, CSS, and JS. After some practice, we committed to working on the website daily. We met either online or in person to collaborate as a team. While building the website, we found YouTube tutorials helpful. The most challenging parts included understanding the box-container concept, distinguishing between margin and padding, and mastering position and display properties.

2. Practice

After learning the basics, we began practicing by writing code for simple web pages. These exercises involved using HTML to structure content, CSS to style elements visually, and JavaScript to add interactivity. This hands-on practice helped us become more comfortable with arranging and organizing website elements effectively.

3. Implementation

During the implementation phase, we focused on designing our pages. We began by sketching the layouts, then used those sketches as a guide for coding. This approach made the coding process much more efficient, as it helped us avoid rewriting the same code multiple times.
